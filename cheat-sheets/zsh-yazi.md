# Zsh Cheat Sheet

Repo-specific shell shortcuts from [`zsh/.zsh_aliases`](../zsh/.zsh_aliases).

## Reload Shell

| Command | Action |
|-----|--------|
| `reload` | Re-source `~/.zshrc` from anywhere |

## Directory Listing

| Command | Action |
|-----|--------|
| `ls` | `eza` list with directories first, icons, and color |
| `l` | One entry per line |
| `la` | Long list including hidden files |
| `ll` | Detailed long list with Git status |
| `lt` | Newest-first paged listing |
| `lta` | Hidden-files version of newest-first paged listing |

## Navigation

| Command | Action |
|-----|--------|
| `cd <dir>` | Uses `zoxide` via alias to jump by frecency |
| `..` | Go up one directory |
| `...` | Go up two directories |
| `....` | Go up three directories |
| `cdp` | Go to `~/Projects` |
| `dots` | Go to `~/dotfiles` |
